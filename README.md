# HTML-BLEserial
Webpage terminal / serial monitor to connect to a serial UART device

To control microprocessors via a serial port or Bluetooth a serial terminal is used.

That is a program that send and receive ASCII characters

When a BLE device using the TI CC2511 (HM-10, HM-16, JDY modules) or a Nordic nRF52 chipsets (all ESP32's) app on a cellular like the iPhone or Androis phones can be installed and used

Since 2026  browsers like chrome and Edge can be used to simulate the serial terminal via BLE.

Check Chrome or Edge flags
Type  in the url locator of chrome: chrome://flags or in Edge:  edge://flags and search for bluetooth — make sure nothing is disabled there. On some versions #enable-web-bluetooth needs to be explicitly enabled.
